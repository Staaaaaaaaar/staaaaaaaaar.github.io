第十三章：随机信号

About 1683 wordsAbout 6 min

2026-01-29

Note

随机信号是实际中广泛存在的一类信号，其数学本质是一族依赖时间指标的随机变量。本章从定义出发，讨论其概率描述、特征函数、功率谱、遍历性；再重点研究平稳信号与线性随机信号（ARMA模型），为后续随机信号处理（如维纳滤波、谱估计等）奠定理论基础。

§1 随机信号

1.1 定义

设 TTT 是一个指标集（如 R,Z,Z2\mathbb{R}, \mathbb{Z}, \mathbb{Z}^2R,Z,Z2），若对任意 t∈Tt \in Tt∈T，X(t)X(t)X(t) 是一个随机变量，则称 X(t)X(t)X(t) 为定义在 TTT 上的随机信号（或随机过程、随机场）。

常见类型：

  • 连续时间随机信号：T=RT = \mathbb{R}T=R
  • 离散时间随机信号（时间序列）：T=ZT = \mathbb{Z}T=Z
  • 二维离散随机场（如图像）：T=Z2T = \mathbb{Z}^2T=Z2 或 T=ZM×ZNT = \mathbb{Z}_M \times \mathbb{Z}_NT=ZM​×ZN​

1.2 随机信号的概率分布

对任意 n≥1n \ge 1n≥1、t1,…,tn∈Tt_1, \dots, t_n \in Tt1​,…,tn​∈T、实数 x1,…,xnx_1, \dots, x_nx1​,…,xn​，联合分布函数定义为：

P(X(t1)≤x1, …, X(tn)≤xn)=F(x1,t1;… ;xn,tn)P\left(X(t_1) \le x_1,\, \dots,\, X(t_n) \le x_n\right) = F(x_1, t_1; \dots; x_n, t_n) P(X(t1​)≤x1​,…,X(tn​)≤xn​)=F(x1​,t1​;…;xn​,tn​)

称为 X(t)X(t)X(t) 的有限维分布族，它完全刻画了随机信号的统计性质。

由于完整分布难以获得，常通过特征数描述随机信号：

  • kkk 阶原点矩：E[Xk(t)]\mathbb{E}[X^k(t)]E[Xk(t)]
  • kkk 阶中心矩：E[(X(t)−EX(t))k]\mathbb{E}\big[(X(t) - \mathbb{E}X(t))^k\big]E[(X(t)−EX(t))k]
  • 自相关函数：rXX(t,s)=E[X(t)X(s)]r_{XX}(t, s) = \mathbb{E}[X(t) X(s)]rXX​(t,s)=E[X(t)X(s)]

1.3 几类重要随机信号

严平稳随机信号

若指标集 TTT 对加减封闭（如 Z\mathbb{Z}Z），且对任意 t,t1,…,tn∈Tt, t_1, \dots, t_n \in Tt,t1​,…,tn​∈T，有：

(X(t1),…,X(tn))=d(X(t1+t),…,X(tn+t))\big(X(t_1), \dots, X(t_n)\big) \overset{d}{=} \big(X(t_1 + t), \dots, X(t_n + t)\big) (X(t1​),…,X(tn​))=d(X(t1​+t),…,X(tn​+t))

即任意有限维分布不随时间平移而改变，则称 X(t)X(t)X(t) 为严平稳（strict-sense stationary, SSS）。

宽平稳随机信号

若对任意 t,s∈Tt, s \in Tt,s∈T，满足：

{E[X(t)]=mX=常数,E[X(t)X(s)]=rXX(t−s)=rXX(τ),τ=t−s\begin{cases} \mathbb{E}[X(t)] = m_X = \text{常数}, \\ \mathbb{E}[X(t) X(s)] = r_{XX}(t - s) = r_{XX}(\tau), \quad \tau = t - s \end{cases} {E[X(t)]=mX​=常数,E[X(t)X(s)]=rXX​(t−s)=rXX​(τ),τ=t−s​

则称 X(t)X(t)X(t) 为宽平稳（wide-sense stationary, WSS）。

Note

严平稳 ⇒\Rightarrow⇒ 宽平稳（若二阶矩存在），反之一般不成立。

离散白噪声信号

若 X(t), t∈ZX(t),\, t \in \mathbb{Z}X(t),t∈Z 满足：

E[X(t)]=0,E[X(t)X(s)]=σ2δ(t−s)\mathbb{E}[X(t)] = 0,\quad \mathbb{E}[X(t) X(s)] = \sigma^2 \delta(t - s) E[X(t)]=0,E[X(t)X(s)]=σ2δ(t−s)

其中 δ\deltaδ 为离散 δ\deltaδ 函数，则称 X(t)X(t)X(t) 为离散白噪声。

1.4 随机信号的能量与功率谱

  • 瞬时功率：E[X2(t)]\mathbb{E}[X^2(t)]E[X2(t)]
  • 平均功率（离散情形）：

    P=lim⁡N→∞12N+1∑n=−NNE[X2(n)]P = \lim_{N \to \infty} \frac{1}{2N+1} \sum_{n=-N}^{N} \mathbb{E}[X^2(n)] P=N→∞lim​2N+11​n=−N∑N​E[X2(n)]

  • 功率谱（离散情形）：

    GXX(ω)=lim⁡N→∞E[12N+1∣∑n=−NNX(n)e−inω∣2]G_{XX}(\omega) = \lim_{N \to \infty} \mathbb{E}\left[ \frac{1}{2N+1} \left| \sum_{n=-N}^{N} X(n) e^{-i n \omega} \right|^2 \right] GXX​(ω)=N→∞lim​E​2N+11​​n=−N∑N​X(n)e−inω​2​

对宽平稳信号，可进一步简化（见 §2）。

1.5 随机信号的遍历性

若对任意函数 V(⋅)V(\cdot)V(⋅)（如 V(x)=x,x2V(x) = x, x^2V(x)=x,x2 等），满足：

E[V(X(t))]=lim⁡N→∞12N+1∑k=−NNV(X(t+k))\mathbb{E}[V(X(t))] = \lim_{N \to \infty} \frac{1}{2N+1} \sum_{k=-N}^{N} V\big(X(t+k)\big) E[V(X(t))]=N→∞lim​2N+11​k=−N∑N​V(X(t+k))

则称 X(t)X(t)X(t) 具有遍历性（ergodicity），即集合平均 = 时间平均。

意义：只需单次观测即可估计统计特征（如均值、自相关），是工程实践的重要基础。

§2 平稳信号和线性随机信号

Note

本节聚焦离散宽平稳信号（即平稳时间序列），并引入最重要的线性随机信号模型——ARMA 模型。

2.1 平稳信号

设实离散随机信号 X(t), t∈ZX(t),\, t \in \mathbb{Z}X(t),t∈Z，若满足：

E[X(t)]=mX,E[X(t)X(s)]=rXX(t−s)\mathbb{E}[X(t)] = m_X,\quad \mathbb{E}[X(t) X(s)] = r_{XX}(t - s) E[X(t)]=mX​,E[X(t)X(s)]=rXX​(t−s)

则称 X(t)X(t)X(t) 为平稳信号。

  • 自相关函数：rXX(n)=E[X(t+n)X(t)]r_{XX}(n) = \mathbb{E}[X(t + n) X(t)]rXX​(n)=E[X(t+n)X(t)]
  • 自协方差函数：cXX(n)=E[(X(t+n)−mX)(X(t)−mX)]c_{XX}(n) = \mathbb{E}\big[(X(t + n) - m_X)(X(t) - m_X)\big]cXX​(n)=E[(X(t+n)−mX​)(X(t)−mX​)]
    • 显然 cXX(n)=rXX(n)−mX2c_{XX}(n) = r_{XX}(n) - m_X^2cXX​(n)=rXX​(n)−mX2​

对两个平稳信号 X(t),Y(t)X(t), Y(t)X(t),Y(t)：

  • 互相关函数：rXY(n)=E[X(t+n)Y(t)]r_{XY}(n) = \mathbb{E}[X(t + n) Y(t)]rXY​(n)=E[X(t+n)Y(t)]
  • 互协方差函数：cXY(n)=rXY(n)−mXmYc_{XY}(n) = r_{XY}(n) - m_X m_YcXY​(n)=rXY​(n)−mX​mY​

  • 平均功率：

    P=rXX(0)P = r_{XX}(0) P=rXX​(0)

    GXX(ω)=∑n=−∞∞rXX(n)e−inω,ω∈[−π,π]G_{XX}(\omega) = \sum_{n = -\infty}^{\infty} r_{XX}(n) e^{-i n \omega}, \quad \omega \in [-\pi, \pi] GXX​(ω)=n=−∞∑∞​rXX​(n)e−inω,ω∈[−π,π]

    且有：

    rXX(n)=12π∫−ππGXX(ω)einω dωr_{XX}(n) = \frac{1}{2\pi} \int_{-\pi}^{\pi} G_{XX}(\omega) e^{i n \omega} \, d\omega rXX​(n)=2π1​∫−ππ​GXX​(ω)einωdω

2.3 平稳信号的滤波

设滤波器脉冲响应为 h(t)h(t)h(t)（实、能量有限），频谱为 H(ω)=∑th(t)e−itωH(\omega) = \sum_t h(t) e^{-i t \omega}H(ω)=∑t​h(t)e−itω。

对平稳输入 X(t)X(t)X(t)，输出 Y(t)=h∗X(t)Y(t) = h * X(t)Y(t)=h∗X(t) 仍为平稳信号，且：

  • 均值：

    mY=mX∑th(t)m_Y = m_X \sum_{t} h(t) mY​=mX​t∑​h(t)

  • 自相关：

    rYY(n)=rhh(n)∗rXX(n)r_{YY}(n) = r_{hh}(n) * r_{XX}(n) rYY​(n)=rhh​(n)∗rXX​(n)

    其中 rhh(n)=h(n)∗h(−n)r_{hh}(n) = h(n) * h(-n)rhh​(n)=h(n)∗h(−n) 为滤波器自相关。

  • 功率谱：

    GYY(ω)=∣H(ω)∣2⋅GXX(ω)G_{YY}(\omega) = |H(\omega)|^2 \cdot G_{XX}(\omega) GYY​(ω)=∣H(ω)∣2⋅GXX​(ω)

    即输出功率谱 = 滤波器增益平方 × 输入功率谱。
