第四章：线性时不变滤波器与系统

About 1499 wordsAbout 5 min

2026-01-29

Note

本章核心：定义线性时不变系统及其基本性质；深入讨论系统的因果性、稳定性及组合方式（串联、并联、反馈）；重点阐述有理系统（递归滤波器）的Z变换表示与差分方程解法。

§1 线性时不变系统及其时间响应函数

1. 系统的一般定义

系统是一种将输入信号 x(n)x(n)x(n) 映射为输出信号 y(n)y(n)y(n) 的装置或数学模型。

y(n)=Tx(n)y(n)=Tx(n) y(n)=Tx(n)

2. 线性时不变系统

  • 线性系统：满足叠加原理的系统。若输入 x1(t),x2(t)x_1(t), x_2(t)x1​(t),x2​(t) 对应输出 y1(t),y2(t)y_1(t), y_2(t)y1​(t),y2​(t)，则输入 ax1(t)+bx2(t)ax_1(t) + bx_2(t)ax1​(t)+bx2​(t) 对应输出 ay1(t)+by2(t)ay_1(t) + by_2(t)ay1​(t)+by2​(t)。

  • 时不变系统：系统的特性不随时间变化。若输入 x(t)x(t)x(t) 对应输出 y(t)y(t)y(t)，则输入 x(t−t0)x(t - t_0)x(t−t0​) 对应输出 y(t−t0)y(t - t_0)y(t−t0​)。

    Note

    时不变性质表明，输出信号的波形不随输入信号的延迟而改变，并且输出信号的延迟与输入信号的延迟是同步的。

3. 线性时不变系统的时间响应函数

当输入为单位冲激函数 δ(t)\delta(t)δ(t) 时，系统的输出称为该系统的时间响应函数，记作 h(t)h(t)h(t)。

h(n)=Tδ(n)h(n) = T \delta(n) h(n)=Tδ(n)

相应地，线性时不变系统 TTT 的频率响应函数和 Z 变换为

H(ω)=∑n=−∞+∞h(n)e−inωH(\omega)=\sum_{n=-\infty}^{+\infty}h(n)e^{-in\omega} H(ω)=n=−∞∑+∞​h(n)e−inω

H(Z)=∑n=−∞+∞h(n)ZnH(Z)=\sum_{n=-\infty}^{+\infty}h(n)Z^{n} H(Z)=n=−∞∑+∞​h(n)Zn

4. 线性时不变系统的输出信号

输出信号 y(n)y(n)y(n) 与输入信号 x(n)x(n)x(n) 和时间响应函数 h(n)h(n)h(n) 的关系为：

y(n)=x(n)∗h(n)y(n)=x(n)*h(n) y(n)=x(n)∗h(n)

H(Z)=Y(Z)X(Z)H(Z)=\frac{Y(Z)}{X(Z)} H(Z)=X(Z)Y(Z)​

§2 线性时不变系统的因果性和稳定性

1. 因果性

  • 连续系统：若系统的时间响应函数 h(t)h(t)h(t) 满足 h(t)=0h(t) = 0h(t)=0 当 t<0t < 0t<0，则称该系统为因果系统或物理可实现系统。
  • 离散系统：若系统的时间响应函数 h(n)h(n)h(n) 满足 h(n)=0h(n) = 0h(n)=0 当 n<0n < 0n<0，则称该系统为因果系统或物理可实现系统。

2. 稳定性

  • 有界输入有界输出（BIBO）稳定性：若系统对任何有界输入信号 x(n)x(n)x(n)（即存在常数 CCC 使得 ∣x(n)∣≤C|x(n)| \le C∣x(n)∣≤C）产生有界输出信号 y(n)y(n)y(n)，则称该系统具有有界输入有界输出稳定性。

定理 1

线性时不变系统 h(n)h(n)h(n) 具有 BIBO 稳定性的充分必要条件是：

∑n=−∞∞∣h(n)∣<+∞\sum_{n=-\infty}^{\infty} |h(n)| < +\infty n=−∞∑∞​∣h(n)∣<+∞

  • 能量有限稳定性：若系统对任何能量有限的输入信号 x(n)x(n)x(n)（即 ∑n=−∞+∞x2(n)<+∞\sum_{n=-\infty}^{+\infty}x^2(n) < +\infty∑n=−∞+∞​x2(n)<+∞）产生能量有限输出信号 y(n)y(n)y(n)，则称该系统具有能量有限稳定性。

定理 2

线性时不变系统 h(n)h(n)h(n) 具有能量有限稳定性的充分必要条件是：存在一个正数 CCC 使得

∣H(ω)∣=∣∑n=−∞+∞h(n)e−inω∣≤C|H(\omega)|=\bigg|\sum_{n=-\infty}^{+\infty}h(n)e^{-in\omega}\bigg|\le C ∣H(ω)∣=​n=−∞∑+∞​h(n)e−inω​≤C

§3 系统的组合：串联、并联及反馈

1. 串联组合

定义：两个系统 T1T_1T1​ 和 T2T_2T2​ 串联，指第一个系统的输出作为第二个系统的输入。

时间响应函数：若 T1T_1T1​ 的时间响应函数为 h1(n)h_1(n)h1​(n)，T2T_2T2​ 的时间响应函数为 h2(n)h_2(n)h2​(n)，则串联后系统的总时间响应函数为：

h(n)=h1(n)∗h2(n)h(n) = h_1(n) * h_2(n) h(n)=h1​(n)∗h2​(n)

Z 变换关系：若 H1(Z),H2(Z)H_1(Z), H_2(Z)H1​(Z),H2​(Z) 分别为 T1,T2T_1, T_2T1​,T2​ 的 Z 变换，则总系统 Z 变换为：

H(Z)=H1(Z)⋅H2(Z)H(Z) = H_1(Z) \cdot H_2(Z) H(Z)=H1​(Z)⋅H2​(Z)

2. 并联组合

定义：两个系统 T1T_1T1​ 和 T2T_2T2​ 并联，指它们共享同一个输入，输出相加。

时间响应函数：若 T1T_1T1​ 的时间响应函数为 h1(n)h_1(n)h1​(n)，T2T_2T2​ 的时间响应函数为 h2(n)h_2(n)h2​(n)，则并联后系统的总时间响应函数为：

h(n)=h1(n)+h2(n)h(n) = h_1(n) + h_2(n) h(n)=h1​(n)+h2​(n)

Z 变换关系：若 H1(Z),H2(Z)H_1(Z), H_2(Z)H1​(Z),H2​(Z) 分别为 T1,T2T_1, T_2T1​,T2​ 的 Z 变换，则总系统 Z 变换为：

H(Z)=H1(Z)+H2(Z)H(Z) = H_1(Z) + H_2(Z) H(Z)=H1​(Z)+H2​(Z)

3. 反馈组合

定义：系统的输出经过一个子系统后，再加入到输出中去。

时间域关系式

y(t)=x(t)∗h1(t)−y(t)∗h2(t)y(t) = x(t) * h_1(t) - y(t) * h_2(t) y(t)=x(t)∗h1​(t)−y(t)∗h2​(t)

Z 变换关系

Y(Z)=X(Z)H1(Z)−Y(Z)H2(Z)Y(Z) = X(Z) H_1(Z) - Y(Z) H_2(Z) Y(Z)=X(Z)H1​(Z)−Y(Z)H2​(Z)

H(Z)=Y(Z)X(Z)=H1(Z)1+H1(Z)H2(Z)H(Z) = \frac{Y(Z)}{X(Z)} = \frac{H_1(Z)}{1 + H_1(Z) H_2(Z)} H(Z)=X(Z)Y(Z)​=1+H1​(Z)H2​(Z)H1​(Z)​

§4 有理系统及其时间响应函数

1. 有理系统

时间响应函数的 Z 变换 H(Z)H(Z)H(Z) 为有理函数的系统称为有理系统。

H(Z)=B(Z)A(Z),B(Z)=b0+b1Z+⋯+bqZq,A(Z)=1+a1Z+⋯+apZpH(Z) = \frac{B(Z)}{A(Z)}, \quad B(Z) = b_0 + b_1 Z + \cdots + b_q Z^q, \quad A(Z) = 1 + a_1 Z + \cdots + a_p Z^p H(Z)=A(Z)B(Z)​,B(Z)=b0​+b1​Z+⋯+bq​Zq,A(Z)=1+a1​Z+⋯+ap​Zp

有理系统也称为递归系统或递归滤波器，因为其输出依赖于过去的输出值。

y(n)=∑i=0qbix(n−i)−∑j=1pajy(n−j)y(n) = \sum_{i=0}^{q} b_i x(n-i) - \sum_{j=1}^{p} a_j y(n-j) y(n)=i=0∑q​bi​x(n−i)−j=1∑p​aj​y(n−j)

2. 稳定有理系统的时间响应函数

稳定性条件：有理系统稳定的充分必要条件是其分母多项式 A(Z)A(Z)A(Z) 在单位圆内及上无根，即所有极点位于单位圆外。

有理分式 H(Z)H(Z)H(Z) 可分解为

H(Z)=∑l=1q−pdlZl+∑j=1m∑lj=1rjcj(lj)(Z−αj)ljH(Z)=\sum_{l=1}^{q-p}d_lZ^l+\sum_{j=1}^m\sum_{l_j=1}^{r_j}\frac{c_j(l_j)}{(Z-\alpha_j)^{l_j}} H(Z)=l=1∑q−p​dl​Zl+j=1∑m​lj​=1∑rj​​(Z−αj​)lj​cj​(lj​)​

其中 αj\alpha_jαj​ 为分母 A(Z)=1+a1Z+⋯+apZpA(Z)=1 + a_1 Z + \cdots + a_p Z^pA(Z)=1+a1​Z+⋯+ap​Zp 的 rjr_jrj​ 重根，∑j=1mrj=p\sum_{j=1}^mr_j=p∑j=1m​rj​=p。

之后基于 H(Z)H(Z)H(Z) 在单位圆内解析区域的罗朗展开，提取相应系数即得 h(n)h(n)h(n)。

§5 差分方程的单边 Z 变换解法

1. 单边 Z 变换

对因果序列 x(n)x(n)x(n)（即 x(n)=0x(n)=0x(n)=0 当 n<0n<0n<0），其单边 Z 变换定义为：

X(Z)=∑n=0∞x(n)ZnX(Z) = \sum_{n=0}^{\infty} x(n) Z^{n} X(Z)=n=0∑∞​x(n)Zn

2. 解法步骤

  1. 写出差分方程：

    y(n)=∑i=0qbix(n−i)−∑j=1pajy(n−j)y(n) = \sum_{i=0}^{q} b_i x(n-i) - \sum_{j=1}^{p} a_j y(n-j) y(n)=i=0∑q​bi​x(n−i)−j=1∑p​aj​y(n−j)

  2. 取单边 Z 变换：

    Y(Z)=∑i=0qbiZiX(Z)−∑j=1pajZjY(Z)Y(Z) = \sum_{i=0}^{q} b_i Z^{i} X(Z) - \sum_{j=1}^{p} a_j Z^{j} Y(Z) Y(Z)=i=0∑q​bi​ZiX(Z)−j=1∑p​aj​ZjY(Z)

  3. 解出 Y(Z)Y(Z)Y(Z)：

    Y(Z)=B(Z)X(Z)A(Z)Y(Z) = \frac{B(Z) X(Z)}{A(Z)} Y(Z)=A(Z)B(Z)X(Z)​

  4. 逆 Z 变换：通过部分分式展开或查表，得到时间响应函数 y(n)y(n)y(n)。
