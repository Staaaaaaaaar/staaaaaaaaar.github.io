第三章：滤波与褶积，Z变换

About 1628 wordsAbout 5 min

2026-01-29

Note

本章系统介绍数字信号处理中最核心的操作之一——滤波，从频域与时间域两个角度阐明其原理，并引入褶积作为滤波的时间域实现方式。进一步，通过引入Z变换，将离散信号频谱简化为代数形式，为后续系统分析（如线性时不变系统、递归滤波器设计等）奠定数学工具基础。

§1 连续信号的滤波与褶积

1. 滤波问题的提出

实际信号 x(t)x(t)x(t) 通常包含有效信号 s(t)s(t)s(t) 与干扰信号 n(t)n(t)n(t)

x(t)=s(t)+n(t)x(t) = s(t) + n(t) x(t)=s(t)+n(t)

若 s(t)s(t)s(t) 与 n(t)n(t)n(t) 的频谱 S(f),N(f)S(f), N(f)S(f),N(f) 分离（即 S(f)N(f)≡0S(f)N(f) \equiv 0S(f)N(f)≡0），则可设计滤波器频谱 H(f)H(f)H(f)

H(f)={1,S(f)≠00,S(f)=0H(f) = \begin{cases} 1, & S(f) \ne 0 \\ 0, & S(f) = 0 \end{cases} H(f)={1,0,​S(f)=0S(f)=0​

使得输出 Y(f)=H(f)X(f)=S(f)Y(f) = H(f)X(f) = S(f)Y(f)=H(f)X(f)=S(f)，即完全消噪、保留有效信号。

滤波便是将原始信号 x(t)x(t)x(t) 的频谱 X(f)X(f)X(f) 与滤波器频谱 H(f)H(f)H(f) 相乘，得输出频谱 Y(f)=X(f)H(f)Y(f) = X(f)H(f)Y(f)=X(f)H(f) 的过程。一般情形下，S(f)S(f)S(f) 与 N(f)N(f)N(f) 重叠但特性不同，仍可依频谱差异设计 H(f)H(f)H(f)，实现削弱干扰、增强信号的目的。

2. 连续信号的滤波与褶积

设输入信号为 x(t)x(t)x(t)，滤波器频谱为 H(f)H(f)H(f)，对应时间函数为 h(t)h(t)h(t)，输出频谱为 Y(f)=X(f)H(f)Y(f) = X(f)H(f)Y(f)=X(f)H(f)，对应时间函数为 y(t)y(t)y(t)。

通过傅里叶逆变换与积分交换顺序，可得

y(t)=∫−∞∞h(τ)x(t−τ) dτy(t) = \int_{-\infty}^{\infty} h(\tau) x(t - \tau)\, d\tau y(t)=∫−∞∞​h(τ)x(t−τ)dτ

此即 褶积（或卷积）运算，记为

y(t)=x(t)∗h(t)y(t) = x(t) * h(t) y(t)=x(t)∗h(t)

§2 离散信号的滤波与褶积

1. 离散信号的滤波与褶积

对抽样间隔为 Δ\DeltaΔ 的离散信号 x(nΔ)x(n\Delta)x(nΔ)，滤波因子 h(nΔ)h(n\Delta)h(nΔ)，其频谱分别为 XΔ(f),HΔ(f)X_\Delta(f), H_\Delta(f)XΔ​(f),HΔ​(f)。若 x(t),h(t)x(t), h(t)x(t),h(t) 均有截频 fcf_cfc​，且抽样满足 1/(2Δ)≥fc1/(2\Delta) \ge f_c1/(2Δ)≥fc​，则离散滤波可精确实现连续滤波。

离散滤波关系为：

YΔ(f)=XΔ(f)HΔ(f)Y_\Delta(f) = X_\Delta(f) H_\Delta(f) YΔ​(f)=XΔ​(f)HΔ​(f)

对应时域关系（褶积）为：

y(nΔ)=Δ∑τ=−∞∞h(τΔ)x((n−τ)Δ)y(n\Delta) = \Delta \sum_{\tau = -\infty}^{\infty} h(\tau\Delta) x\big((n - \tau)\Delta\big) y(nΔ)=Δτ=−∞∑∞​h(τΔ)x((n−τ)Δ)

记为：

y(nΔ)=x(nΔ)∗h(nΔ)y(n\Delta) = x(n\Delta) * h(n\Delta) y(nΔ)=x(nΔ)∗h(nΔ)

2. 褶积的直观意义

将离散褶积改写为：

y(nΔ)=Δ∑τ=−∞∞h(−τΔ)x((n+τ)Δ)y(n\Delta) = \Delta \sum_{\tau = -\infty}^{\infty} h(-\tau\Delta) x\big((n + \tau)\Delta\big) y(nΔ)=Δτ=−∞∑∞​h(−τΔ)x((n+τ)Δ)

几何解释：

  1. 将 h(⋅)h(\cdot)h(⋅) 作翻褶（h(τ)→h(−τ)h(\tau) \to h(-\tau)h(τ)→h(−τ)）
  2. 将翻褶后的 h(−τ)h(-\tau)h(−τ) 与 x(τ)x(\tau)x(τ) 对齐于 nnn 处
  3. 逐点相乘并求和（乘积累加），即得 y(nΔ)y(n\Delta)y(nΔ)
  4. 移动对齐点 n→n+1n \to n+1n→n+1，重复即可得整个输出序列

此即“褶而后积”名称来源：先翻褶，再对应相乘累加。
