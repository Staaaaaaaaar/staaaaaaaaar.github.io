地震定位

约 1020 字大约 3 分钟

2026-06-06

在地震定位中，震源 (hypocenter) 被看成一个点，震源在地表的垂直投影叫震中 (epicenter)。地震定位就是为了确定震源的物理位置 (x,y,z)(x,y,z)(x,y,z) 以及发震时刻 ttt。

我们假设：

  • 已知真实的地球速度模型
  • 已知观测台站坐标以及到时数据

震中位置估算

设介质速度均匀，震源坐标为 (x,y,z)(x, y, z)(x,y,z)，观测台站坐标为 (xα,yα,zα)(x_\alpha, y_\alpha, z_\alpha)(xα​,yα​,zα​)，发震时刻为 ttt，则理论到时为

tα=1v(x−xα)2+(y−yα)2+(z−zα)2+tt_\alpha = \frac{1}{v} \sqrt{(x - x_\alpha)^2 + (y - y_\alpha)^2 + (z - z_\alpha)^2} + t tα​=v1​(x−xα​)2+(y−yα​)2+(z−zα​)2​+t

以地表观测台站为坐标原点，台站与震中得连线为 XXX 轴，则

tP=1αx2+z2+tt^P = \frac{1}{\alpha} \sqrt{x^2 + z^2} + t tP=α1​x2+z2​+t

tS=1βx2+z2+tt^S = \frac{1}{\beta} \sqrt{x^2 + z^2} + t tS=β1​x2+z2​+t

S 波与 P 波的到时差为

tS−tP=(1β−1α)x2+z2t^S - t^P = (\frac{1}{\beta}-\frac{1}{\alpha}) \sqrt{x^2 + z^2} tS−tP=(β1​−α1​)x2+z2​

假设 z/x≪1z/x \ll 1z/x≪1，则有

x≈αβα−β(tS−tP)x \approx \frac{\alpha\beta}{\alpha-\beta} (t^S - t^P) x≈α−βαβ​(tS−tP)

上式表明如果能获得 S 波与 P 波的到时差，就能估算出震中距。如果有三个以上的台站，就可以粗略估计出震中位置。
