# 第零章 绪论

## 1 弹性理论概述

- **定义** 
  弹性理论（弹性力学）研究**载荷作用下弹性体**的内力状态与变形规律。  

  - **载荷类型**：机械力、温度、电磁力等；
  - **弹性体**：卸载后完全恢复初始形状与尺寸的理想化模型。

- **发展历程简表**

  | 时间     | 人物       | 贡献                             |
  | -------- | ---------- | -------------------------------- |
  | 1638     | 伽利略     | 梁的弯曲问题（建筑工程需求）     |
  | 1678     | 胡克       | 提出胡克定律（应力 ∝ 应变）      |
  | 1821–22  | 纳维、柯西 | 建立弹性理论普遍方程（数学基础） |
  | 1908     | 里茨       | 能量原理直接解法（近似解开端）   |
  | 1915     | 迦辽金     | 迦辽金法（变分近似）             |
  | 20世纪后 | —          | 计算力学（FEM、BEM等）兴起       |

- **学科地位与作用**

  - 是**固体力学核心基础课**，承上（理论/材料力学）启下（塑性、板壳、断裂力学等）；
  - 与**数学、物理**联系紧密，亦是**连续介质力学、计算力学**的理论基础。


## 2 弹性理论的基本假设

| 假设名称         | 内容                                                         | 物理/数学意义                          |
| ---------------- | ------------------------------------------------------------ | -------------------------------------- |
| **连续性假设**   | 物体为密实连续介质，变形过程始终保持连续                     | 引入连续函数描述物理量，可用微积分工具 |
| **弹性假设**     | 变形与载荷一一对应，卸载后完全恢复，细分为：<br>- **线弹性**：应力–应变线性（胡克定律）<br>- **非线性弹性**：材料非线性或几何非线性 |                                        |
| **均匀性假设**   | 各点材料性质相同                                             | 弹性常数 $E, \nu, G$ 为空间常量        |
| **各向同性假设** | 同一点各方向性质相同                                         | 应力/应变主方向重合；弹性常数仅2个独立 |
| **无初应力假设** | 加载前为自然无应力状态                                       | 忽略残余应力与装配应力                 |


## 3 载荷分类

### 按物理性质分类

| 类型                 | 举例                 | 引起效应                       |
| -------------------- | -------------------- | ------------------------------ |
| **第一类（力型）**   | 重力、机械力、电磁力 | 先引起内力，再引起变形         |
| **第二类（变形型）** | 温度、中子辐照       | 先引起变形，若受约束才产生内力 |

### 按作用域分类

| 名称                              | 定义               | 数学描述                                                    |
| --------------------------------- | ------------------ | ----------------------------------------------------------- |
| **体积力（体力）** $\mathbf{f}$ | 作用于物体内部体积 | $f_i = \lim_{\Delta V \to 0} \frac{\Delta F_i}{\Delta V}$ |
| **表面力（面力）** $\mathbf{p}$ | 作用于物体表面     | $p_i = \lim_{\Delta S \to 0} \frac{\Delta G_i}{\Delta S}$ |
